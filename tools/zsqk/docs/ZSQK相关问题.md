# ZSQK 相关问题

## 两张锁货表 `gy_mall_lock_sku` `gy_sku_lock`

### 基础信息

| 表名             | 说明       |
| ---------------- | ---------- |
| gy_mall_lock_sku | 商城锁货表 |
| gy_sku_lock      | 全局锁货表 |

### 两张锁货表的区别

- 全局锁货表仅记录了 `锁货类型`, `skuID`, `锁货数量`, `锁货时间`;
- 商城锁货表 除了以上字段，还记录了 `仓库ID`, `货品ID`, `相关单号`, 赠品相关数据;
