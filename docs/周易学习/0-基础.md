## 爻位

  ```mermaid
  graph TD
    B["
    上九 fa:fa-minus fa:fa-minus fa:fa-minus
    九五 fa:fa-minus fa:fa-minus fa:fa-minus
    九四 fa:fa-minus fa:fa-minus fa:fa-minus
    九三 fa:fa-minus fa:fa-minus fa:fa-minus
    九二 fa:fa-minus fa:fa-minus fa:fa-minus
    初九 fa:fa-minus fa:fa-minus fa:fa-minus
    "]
  ```

`初九` `九三` `九五` 为奇数, 奇数的爻位是阳爻的爻位;  
`九二` `九四` `上九` 为偶数, 偶数的爻位是阴爻的爻位.  

卦中奇数的爻位正好是阳爻，这叫 `当位`;  
如果是阴爻占了奇数的爻位，这叫 `位不当`;  
一般卦中 **当位吉, 不当位咎**

`九二` 是下卦的中位;  
`九五` 是上卦的中位;  
中位一般都很吉利.  

有的爻与爻之间是遥相呼应的, 它是指**上卦与下卦之间的呼应**.  
初与四, 二与五, 三与上, 这都是遥相呼应的;  
但条件必须是`一阴一阳才相应, 同性则不相应`  
如: 同为阳爻或同为阴爻, 则不相应.
