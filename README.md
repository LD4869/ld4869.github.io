# ld4869.github.io

## Vue 项目搭建流程

已有 **git** | **node** | **npm**

1. 从 github 上克隆项目

   ```bash
   git clone https://github.com/ld4869/ld4869.github.io.git
   ```

2. 进入项目目录

   ```bash
   cd ld4869.github.io
   ```

3. 初始化 **npm** 项目
  
   ```bash
   npm init -y  & :: 生成 package.json 文件
   ```

4. 安装依赖

   ```bash
   npm install -g @vue/cli & :: 安装 vue-cli
   vue --version & :: 查看 vue-cli 版本
   ```

5. 创建 **Vue** 项目

   ```bash
   vue create vue-app & :: 创建 vue 项目
   ```

## 常用命令

```bash
npm run serve & :: 运行项目
npm run build & :: 打包项目
npm run deploy & :: 部署项目
```
